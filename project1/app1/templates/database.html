¿QUÉ ES PLAN&PARK?

El desarrollo de esta aplicación pretende dar una ayuda a las personas que tengan minusvalía a la hora de ser conductores o pasajeros dentro de un vehículo. Dichas personas necesitan de plazas reservadas que tienen una mayor amplitud. Este proyecto se plantea pues como una aplicación que ofrezca recomendaciones con planes de ocio con accesos habilitados para personas con minusvalía, ofreciendo datos de las plazas de aparcamiento que necesitan. Estos datos contendrán información respecto a la disponibilidad de la plaza (ocupada/libre), tiempo de espera para poder ocupar la plaza. Adicionalmente, se pretende desarrollar un sistema de pago de tickets de aparcamiento mediante la aplicación, en un entorno móvil. La aplicación ofrecerá recomendaciones de centros de ocio a usuarios que no tengan minusvalía.
